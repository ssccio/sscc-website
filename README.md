# SSCC Cloud Consulting Website

Marketing website for Secure Systems Cloud Consulting (SSCC).

## Overview

SSCC provides enterprise cloud consulting services including:
- AWS architecture, migration, and optimization
- Kubernetes and container orchestration
- CI/CD pipeline implementation
- OpenShift deployment and management
- Terraform infrastructure as code
- Serverless architecture design

## Tech Stack

- **Framework**: React + TypeScript
- **Build**: Vite
- **Styling**: Tailwind CSS
- **Components**: shadcn/ui

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## Design

- **Color scheme**: Charcoal (#1a1a1a) + Burnt Orange (#E85D04)
- **Style**: Dark theme, industrial/infrastructure aesthetic
- **Target audience**: CTOs, VPs Engineering, Dev Leads at mid-market companies

## Structure

- `/` - Homepage with hero, services overview, stats
- `/services` - Detailed service offerings (AWS, K8s, CI/CD, etc.)
- `/case-studies` - Client success stories with metrics
- `/resources` - Whitepapers, guides, downloadable content
- `/blog` - Technical articles and insights
- `/about` - Company mission and values
- `/contact` - Contact form and consultation booking
